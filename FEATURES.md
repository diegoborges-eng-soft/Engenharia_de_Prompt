# Funcionalidades da MENTE COLETIVA

## 1. Autenticação

### Registro e Login
- Crie uma conta com email e senha
- Interface futurista com validações
- Sessão persistente

```
Fluxo:
1. Clique em "Não tem conta? Crie uma"
2. Digite nome de usuário, email e senha
3. Clique em "Criar conta"
4. Pronto! Você já pode compartilhar pensamentos
```

## 2. Compartilhamento de Pensamentos

### Campo de Pensamento
- Limite de 280 caracteres
- Interface gigante e futurista
- Feedback visual em tempo real
- Análise automática de emoção

```
Como usar:
1. Escreva seu pensamento no campo central
2. Clique em "Compartilhar"
3. Sistema analisa automaticamente a emoção
4. Pensamento aparece no Feed Neural
```

## 3. Feed Neural

### Visualização em Realtime
- Veja pensamentos conforme são compartilhados
- Animações suaves e fluidas
- Mostra perfil do autor e timestamp
- Exibe emoção detectada
- Botão para conectar com autores

```
Recursos:
- Auto-refresh a cada novo pensamento
- Scroll suave
- 50 pensamentos mais recentes
- Sem likes ou números públicos
```

## 4. Compatibilidade Cognitiva (Matching)

### Encontre Pessoas Similares
- Sistema analisa seus pensamentos e os de outros
- Calcula compatibilidade baseada em emoções compartilhadas
- Mostra top 5 pessoas mais compatíveis com você
- Exibe percentual de compatibilidade

```
Como funciona:
1. Quando você compartilha pensamentos, o sistema processa
2. Detecta suas emoções predominantes
3. Encontra outros usuários com padrões similares
4. Cria conexões automáticas
5. Mostra grau de compatibilidade (0-100%)
```

### Insights Emocionais
- Visualiza seu perfil emocional
- Mostra top 3 emoções
- Percentual de cada emoção
- Gráficos em barras animadas

## 5. Salas Automáticas

### Grupos por Tema Emocional
- Sistema cria automaticamente salas baseadas em emoções
- Salas são temporárias (válidas por 24h)
- Participação opcional
- Chat em tempo real

```
Salas disponíveis por emoção:
- Celebrando momentos felizes (Alegria)
- Processando sentimentos (Tristeza)
- Canalizando energia (Raiva)
- Enfrentando medos juntos (Medo)
- Descobertas inesperadas (Surpresa)
- Meditação coletiva (Calma)
- Buscando clareza (Confusão)
- Sonhando juntos (Esperança)
- Questionando o impossível (Dúvida)
- Compartilhando bênçãos (Gratidão)
```

### Como Participar
1. Clique em "Entrar" na sala desejada
2. Você agora é membro da sala
3. Pode ver mensagens e participar
4. Status muda para "Dentro"

## 7. Segurança e Privacidade

### Sem Rastreamento Público
- Sem contadores de visualizações
- Sem sistema de likes
- Sem count de followers
- Sem ranking de usuários
- Foco em conexão genuína

### Segurança de Dados
- Autenticação via Supabase
- Row Level Security em todas as tabelas
- Cada usuário vê apenas dados que pode acessar
- Pensamentos são seus e de acesso público
- Conexões são privadas entre duas pessoas

## 8. Realtime

### Atualizações em Tempo Real

- Novas salas surgem conforme criadas
- Novas conexões são calculadas automaticamente
- Chat ao vivo nas salas
